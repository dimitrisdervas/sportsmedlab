TRIM

trim Returns a slice of a passed string with all leading and trailing characters from cutset removed.

strings.TrimLeft Returns a slice of a given string with all leading characters contained in the cutset removed.

strings.TrimPrefix Returns a given string s without the provided leading prefix string. If s doesn’t start with prefix, s is returned unchanged.

strings.TrimRight Returns a slice of a given string with all trailing characters contained in the cutset removed.

strings.TrimSuffix Returns a given string s without the provided trailing suffix string. If s doesn’t end with suffix, s is returned unchanged.

substrExtracts parts of a string from a specified character’s position and returns the specified number of characters.

Path and URL

path.BaseBase returns the last element of a path.

string variable: {{{{$stringPlainpath}}}} result: {{ $path }}
{{ path.Base {{$stringPlainpath}} }}
{{ path.Base $path }}

path.DirDir returns all but the last element of a path.

string variable: {{{{$stringPlainpath}}}} result: {{ $path }}
{{ {{$stringPlainpath}} | path.Dir }}
{{ path.Dir $path }}

path.ExtExt returns the file name extension of a path.

string variable: {{{{$stringPlainpath}}}} result: {{ $path }}
{{ {{$stringPlainpath}} | path.Ext }}
{{ $path | path.Ext }}

path.JoinJoin path elements into a single path.

string variable: {{{{$stringPlainpath}}}} result: {{ $path }}
{{ path.Join {{$stringPlainpath}} {{$stringPlainpath}} }}
{{ path.Join $path $path }}

string variable: {{{{$stringPlainpath}}}} result: {{ $path }}
{{ path.Split {{$stringPlainpath}}}}
